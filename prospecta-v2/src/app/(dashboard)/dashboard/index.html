Comment creer les cles API WooCommerce

  1. Va dans WooCommerce > Reglages > Avance > REST API
  2. Clique Ajouter une cle
  3. Description : Cockpit Scoring
  4. Utilisateur : ton compte admin
  5. Permissions : Lecture (suffisant)
  6. Clique Generer la cle API
  7. Copie la Consumer Key (ck_...) et Consumer Secret (cs_...) ici
Important : ton site doit avoir SSL (https://) et les permaliens reglages doivent etre actives (pas en mode simple).

Cloudflare Workers — Accès rapide Afrique

GRATUIT
Serveurs edge à Lagos, Abidjan, Accra — connexion en <20ms depuis l'Afrique de l'Ouest.
1. Va sur workers.cloudflare.com → créer un compte gratuit
2. Workers & Pages → Créer → Déployer le fichier cloudflare-worker.js
3. Settings → Variables → ajouter CF_API_KEY = une clé secrète de ton choix
4. KV → Créer un namespace nommé COCKPIT_KV → le lier au Worker
🔥

Firebase Firestore — Base de données recommandée

GRATUIT
Remplace JSONBin par Google Firebase — plus rapide, plus fiable, gratuit.
1. Va sur console.firebase.google.com → Créer un projet
2. Active Firestore Database → Mode test
3. Paramètres du projet (⚙️) → Tes applications → icône Web → copie projectId et apiKey

Configuration JSONBin.io (base de donnees gratuite)

1. Cree un compte gratuit sur jsonbin.io
2. Cree un nouveau Bin avec ce contenu : {"users":[]}
3. Copie le Bin ID et ta Master Key ci-dessous
